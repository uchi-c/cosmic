# MONO ATELIER Fashion Commerce Prototype

Static responsive e-commerce prototype for a modern fashion brand.

## Features

- Home, shop, product detail, about, and contact views
- Product search, category filter, and price range filter
- Local-storage cart with quantity controls and total calculation
- WhatsApp checkout message generation
- Responsive mobile-first layout
- Placeholder product imagery from Unsplash

## Run

Open `index.html` directly in a browser, or serve the folder with any static server.

```bash
npx serve .
```

Update `WHATSAPP_NUMBER` in `app.js` before deploying.
